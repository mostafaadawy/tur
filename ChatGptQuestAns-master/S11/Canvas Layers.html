The HTML5 <canvas> element itself does not inherently support multiple layers like separate canvases. It represents a single drawing surface and any drawing operations performed on it are directly applied to the entire canvas. If you clear or modify any part of the canvas, it affects the entire canvas.

However, you can achieve the concept of layers by using JavaScript and managing multiple canvas elements stacked on top of each other. Each canvas element represents a separate layer, and you can control the visibility, order, and content of each layer individually.

Here's a simplified example of how you can create multiple layers using separate <canvas> elements: 

html
<!DOCTYPE html>
<html>
<head>
  <title>Canvas Layers Example</title>
  <style>
    canvas {
      position: absolute;
      top: 0;
      left: 0;
    }
  </style>
</head>
<body>
  <canvas id="background" width="500" height="500"></canvas>
  <canvas id="foreground" width="500" height="500"></canvas>

  <script>
    const backgroundCanvas = document.getElementById('background');
    const foregroundCanvas = document.getElementById('foreground');

    const backgroundCtx = backgroundCanvas.getContext('2d');
    const foregroundCtx = foregroundCanvas.getContext('2d');

    // Draw on the background layer
    backgroundCtx.fillStyle = 'green';
    backgroundCtx.fillRect(0, 0, backgroundCanvas.width, backgroundCanvas.height);

    // Draw on the foreground layer
    foregroundCtx.fillStyle = 'red';
    foregroundCtx.fillRect(100, 100, 200, 200);

    // Clear the background layer
    backgroundCtx.clearRect(0, 0, backgroundCanvas.width, backgroundCanvas.height);
  </script>
</body>
</html>

In this example, we have two canvas elements stacked on top of each other using absolute positioning. The background layer is drawn with a green rectangle, while the foreground layer is drawn with a red rectangle. Clearing the background layer using backgroundCtx.clearRect(...) does not affect the foreground layer, and you will see the red rectangle remaining.

By managing multiple canvas elements and their respective contexts, you can create the illusion of layers and have control over individual parts of the drawing surface.
